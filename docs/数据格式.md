# 数据结构

## 近期消息记录界面

### ListItem
```js

{
    id: 1, // 聊天列表项ID
    name: '群聊1', // 名字
    isTop: false, // 是否置顶
    ava: '', // 头像
    status: 0, // 该消息状态 0->屏蔽 1->免打扰 2->正常 3->特别关注
    unreadNum: 10, // 未读的消息数目
    lastInfo: {
        id: 1, // 消息id
        username: 'zhangsan',
        content: 'hihi', // 消息内容
        time: '23-22 19:00', // 消息发送时间
        formId: 3, // 谁发的
        toId: 2 // 发给谁
    }
}

```
